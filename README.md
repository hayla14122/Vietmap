# Vietmap
VietMap – Quỹ đạo Bác Hồ là một bản đồ tương tác trực quan, giúp người xem theo dõi hành trình cuộc đời và hoạt động cách mạng của Chủ tịch Hồ Chí Minh. Dự án sử dụng Leaflet.js để hiển thị bản đồ thế giới, các marker đánh dấu sự kiện quan trọng, cùng polyline snake animation minh họa quãng đường bôn ba.
